Java Intermediate Programs

1. StudentManagementSystem.java     - Demonstrates OOP and ArrayList usage.
2. FileWordCounter.java             - Counts words from a text file using File IO and HashMap.
3. MultithreadedDownloader.java     - Simulates parallel file downloads using multithreading.
4. SimpleNotepad.java               - A basic GUI-based notepad using Java Swing.
5. BankingSystem.java               - Simulates banking transactions using OOP.
6. JDBCLoginValidator.java          - Verifies login using JDBC (requires MySQL setup).
